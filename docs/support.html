PaperPilot PDF Support

If you need help with PaperPilot PDF, please contact the developer using the support email listed on the App Store page.

When reporting an issue, include:

  • Device model
  • iOS version
  • App version
  • The screen where the issue occurred
  • A short screen recording or screenshot if possible

PaperPilot PDF is a simple, fast, on-device PDF scanning app.
